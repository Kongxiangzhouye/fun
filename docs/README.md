# 《万象唤灵》文档索引

单机 Web 放置抽卡项目（`idle-gacha-realm`）：Vite 5 + TypeScript，灵石使用 `decimal.js` 大数，存档在浏览器 `localStorage`。

| 文档 | 说明 |
|------|------|
| [GAME_DESIGN.md](./GAME_DESIGN.md) | **主文档**：资源与循环、各系统玩法、五行灵脉、幻域/装备/灵宠/心法、轮回与飞升、技术要点与源码索引 |

**快速构建**：仓库根目录执行 `npm install`、`npm run dev`（开发）或 `npm run build`（生产构建）。
